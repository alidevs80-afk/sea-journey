# Sea Journey

A small collection of **Seaborn examples** for plotting graphs and visualizing data.  
This repository is part of my learning journey, where I explore how Seaborn makes statistical data visualization easier and more beautiful.

---

## 🌊 Features
- Examples of bar plots, line plots, scatter plots, and heatmaps  
- Built-in dataset usage (tips, iris, etc.) for quick experiments  
- Clean, modern styles with minimal code  
- Focus on statistical visualization and aesthetics
